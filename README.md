# Perfil Topográfico con Peligro

Es un complemento (plugin) para **QGIS** diseñado para la generación de perfiles topográficos dinámicos que integran niveles de peligro geológico. Esta herramienta permite a ingenieros y geólogos visualizar la morfología del terreno y la zonificación de peligro simultáneamente.

## 🚀 Características Principales
* **Segmentación Automática**: Genera perfiles a partir de un DEM segmentados por capas de polígonos de peligro.
* **Gestión de Colores**: Interfaz interactiva para asignar colores específicos a cada categoría de peligro.
* **Interacción Dinámica**: Al mover el cursor en el perfil, se muestra un marcador en el mapa de QGIS para identificar la ubicación geográfica exacta.
* **Escala Real**: Opción para visualizar el perfil en escala 1:1, esencial para análisis geológicos de pendientes.

## 📖 Metodología
Este plugin ha sido desarrollado como una evolución técnica inspirada en las capacidades del plugin **"Terrain Profile"**, optimizado para flujos de trabajo de gestión de riesgos.
